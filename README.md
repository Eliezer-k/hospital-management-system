#Hospital Management System (JavaFX)

Project Overview

This is a desktop application developed using JavaFX and MySQL for managing hospital operations such as patients, doctors, and appointments.

The system provides an interface for efficient hospital data management.

---

Objective

The main objectives of this project are:

- Manage patient records
- Manage doctor information
- Handle appointment scheduling
- Store and retrieve data using a database
- Build a desktop GUI application using JavaFX

---

Tools & Technologies

- Java (JDK)
- JavaFX (GUI)
- MySQL (Database)
- JDBC (Database Connectivity)
- Scene Builder (UI Design)

---

Features

- Doctor management (Add, Update, Delete)
- Patient management
- Appointment scheduling
- Database integration
- User-friendly desktop interface

---

Database

The system uses a MySQL database with tables such as:

- patients
- doctors
- appointments
- users

---

Architecture

- JavaFX for UI layer
- Java controllers for logic
- JDBC for database communication
- MySQL for data storage

---


How to Run the Project

1. Install Java JDK
2. Install MySQL and import database
3. Configure database connection in code
4. Run project from IDE (IntelliJ / Eclipse / NetBeans)
